Week 2 · Days 4–10
WS Producer → Kafka
Build ws_producer.py: connect to Binance wss://stream.binance.com, parse the trade JSON, add event timestamp, push into Kafka with coin as partition key. Add reconnect logic + backoff.
WebSocket Kafka Producer
Week 3 · Days 8–14
Databricks Structured Streaming ⚠️
This is the hardest week. Spark-Kafka connector version issues are real. Budget extra time. Add watermarking for late data, checkpointing for fault tolerance, maxOffsetsPerTrigger for backpressure.
Learn Spark Stream + Window

Delta Lake Medallion Layers
Bronze (raw Kafka data) → Silver (parsed, deduped, typed) → Gold (aggregated, anomaly-flagged). Use MERGE for upserts. Enable time travel. This is your storage story for interviews.
